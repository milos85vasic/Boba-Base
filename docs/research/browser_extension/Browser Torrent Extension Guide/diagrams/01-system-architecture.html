📊 Primary Data Flow — Single Torrent Send

① Content Script scans DOM → ② Magnet links detected → ③ Background SW receives → ④ BobaAPIClient HTTP request
⑤ Boba Service validates → ⑥ qBitTorrent WebUI add → ⑦ Torrent added to client → ⑧ Status returned to Popup

📊 Batch Processing Flow — Tab Groups

① Context menu on Tab Group → ② chrome.tabGroups.query → ③ chrome.tabs.query → ④ Content Script per tab
⑤ Collect all torrents → ⑥ Batch API call → ⑦ All added to qBitTorrent
