⚡ onInstalled
First install: initialize default config, show welcome page, request permissions. Update: show changelog, migrate data.
⚡ onStartup
Browser restart: restore config from storage, re-initialize context menus, check server health, resume pending queue.
⚡ onUpdateAvailable
New version: show update notification, backup current config, schedule update on next browser restart.
