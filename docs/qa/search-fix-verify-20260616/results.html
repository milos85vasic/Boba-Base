2. Multi-word query URL-encoding crash (the “search broken / crashing a lot”)

Root cause (FACT): ~17 nova3 plugins interpolated the RAW query into a request URL; a literal space crashed urllib (URL can't contain control characters). Single-word worked, multi-word silently lost the plugin. Fix: per-plugin URL-encoding (commits dbd3858 7 maintained + da7d709 10 adopted/limetorrents) + honesty classifier plugin_bad_query_encoding (33d90f2).
